📊 Comparative Analysis of Airbnb Market Trends Bangalore Mumbai
📌 Project Overview

This project focuses on analyzing Airbnb data to understand pricing patterns, customer preferences, and location-based trends. The goal is to extract meaningful insights that help in business decision-making and market understanding.

🎯 Objectives
Analyze pricing trends across different locations
Identify factors affecting listing prices
Compare neighborhoods based on demand and cost
Visualize data for better insights


🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib)
Power BI (Dashboard & Visualization)


📂 Dataset Description

The dataset includes:

Listing details (price, room type, location)
Availability and booking information
Host and neighborhood data
🔍 Types of Analysis

1. Descriptive Analysis
Summarizes the dataset using averages, counts, and distributions.

2. Diagnostic Analysis
Explains why certain trends occur (e.g., why some areas have higher prices).

3. Predictive Analysis
Uses patterns in data to estimate future pricing trends.

4. Prescriptive Analysis
Suggests actions based on insights (e.g., best locations to invest).

📊 Key Insights
Certain neighborhoods have significantly higher pricing
Entire homes cost more than shared/private rooms
Price is influenced by location, availability, and demand

📈 Dashboard

Interactive dashboards were created using Power BI to visualize:

Price distribution
Top neighborhoods
Availability trends

📌 Conclusion

This project demonstrates how data analysis can uncover trends in the Airbnb market and support data-driven decisions.
